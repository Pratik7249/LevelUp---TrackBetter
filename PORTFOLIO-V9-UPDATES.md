# Portfolio V9 updates

- Custom growth predictor can include or exclude the latest current portfolio value.
- Predictor now asks for an exact starting month and ending year.
- First-year projection uses only the months from the selected starting month onward.
- Published fund return data is maintained in a separate Fund return analytics section.
- Return periods include 1M, 3M, 6M, 1Y, 3Y, 5Y and All-time.
- Return snapshots are displayed separately from personal portfolio gain, SIP cash flow and XIRR.
- Return snapshot inputs were removed from the Add holding and Update holding forms.
